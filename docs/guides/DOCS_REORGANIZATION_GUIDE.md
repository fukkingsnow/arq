# Организация документации - Пособие

## PowerShell скрипт для автоматического перемещения

Для эффективного перемещения 70+ документов в структурированные папки используйте PowerShell скрипт.

### Структура /docs/

```
/docs/
├── /guides/         # Практические руководства
├── /architecture/   # Архитектура и дизайн
├── /phases/         # По фазам разработки
├── /deployment/     # Развёртывание
├── /operations/     # Операционные наруки
└── /reference/      # Справочные материалы
```

### Инструкции

1. Откройте PowerShell в папке проекта
2. Выполните скрипт
3. Проверьте изменения
4. Выполните git commit
