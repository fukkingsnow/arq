# Индекс файлов для архивирования - Фазы 0-16

## Статус архивирования

Этот документ содержит полный список 35 файлов, относящихся к фазам 0-16 разработки ARQ Backend, которые предназначены для архивирования.

## Причина архивирования

В ходе комплексного аудита документации (Фаза 8) было выявлено, что файлы фаз 0-16 содержат:
- Устаревшие спецификации
- Противоречивые наработки (FastAPI vs NestJS)
- Дублирование информации с современной документацией
- Исторические данные развития проекта

Эти файлы сохраняются в архиве в целях истории развития проекта.
